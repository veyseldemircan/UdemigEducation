# Udemig Education

Welcome to the **Udemig Education** project! A modern, responsive, and user-friendly education platform built with **HTML5**, **SCSS**, and **JavaScript**. This project provides a clean interface for exploring courses, learning about the academy, and interacting with content.

## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Activities/Bullseye.png" width="25" /> Project Overview

Udemig Education enables users to:

- **Explore Courses:** Browse different course categories like Frontend, Backend, and more
- **Modern UI:** Clean and structured layout inspired by real education platforms
- **Responsive Experience:** Fully compatible with all devices
- **Interactive Content:** Smooth hover effects and transitions
- **Dynamic Navigation:** User-friendly navbar and menu interactions
- **Contact Interaction:** Simple and functional contact form

## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Travel%20and%20places/Rocket.png" width="25" /> Features

- **SCSS Architecture:** Modular and reusable styling structure
- **Responsive Design:** Optimized for mobile, tablet, and desktop
- **Grid Layout System:** Organized content using modern layout techniques
- **Hover Animations:** Interactive UI with smooth effects
- **Course Cards:** Structured and styled course components
- **Icon Integration:** Uses **Bootstrap Icons** and **Font Awesome**
- **Navigation System:** Navbar with responsive menu toggle

## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Objects/Hammer%20and%20Wrench.png" width="25" /> Technologies Used

- **HTML5** (Structure)
- **SCSS (Sass)** (Styling)
- **JavaScript** (Interactivity)
- **Bootstrap Icons** (Icons)
- **Font Awesome** (Icons)
